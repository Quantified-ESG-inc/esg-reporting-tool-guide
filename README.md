# ERT 사용자 가이드

## 📖 ERT 사용자 가이드

\*\*ESG Reporting Tool(ERT)\*\*은 복잡한 ESG 데이터를 체계적으로 관리하고, 글로벌 공시 기준에 맞는 보고서를 신속하게 생성할 수 있도록 돕는 전문 솔루션입니다.

이 가이드는 사용자의 역할에 따라 가장 효율적인 ERT 활용 방법을 안내합니다. 본인의 역할에 맞는 가이드를 선택해 주세요.

***

#### 👤 나에게 맞는 가이드 찾기

* **\[유형 1] 처음 시작하는 ESG 담당자** 🐣
  * ESG가 처음이거나, 처음으로 보고서를 발간하는 중소/중견기업 담당자를 위한 단계별 가이드입니다.
* **\[유형 2] 효율을 높이고 싶은 숙련 담당자** 🚀
  * 기존 PDF 보고서 발간 경험이 있으며, 디지털 전환을 통해 업무 효율을 높이고 싶은 담당자용 가이드입니다.
* **\[유형 3] 데이터 작성을 요청받은 협업자** 📝
  * 타 부서에서 지표 입력 및 증빙 업로드를 요청받은 분들을 위한 초간단 가이드입니다.
