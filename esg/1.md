# \[1단계] 초대 수락 및 로그인하기



번거로운 메신저나 이메일 공유 없이도, ERT 한 곳에서 원고 작성부터 동료들의 피드백 확인까지 모든 협업 과정을 스마트하게 관리해 보세요. 나에게 할당된 업무 상태를 업데이트하고 의견을 나누며 완성도 높은 보고서를 함께 만들어갈 수 있습니다.





## 이메일 초대로 시작하기

{% @arcade/embed flowId="7tQ3JcA1qtRn6LMIapcQ" url="https://app.arcade.software/share/7tQ3JcA1qtRn6LMIapcQ" %}

{% stepper %}
{% step %}
#### 초대 메일 확인하기

*   메일함에서 초대장을 찾아주세요.

    * 보낸 사람: `reporting@qesg.co.kr`
    * 메일 제목: `[ESG Reporting Tool] ESG 담당자님으로부터의 Report 작성 초대가 완료되었습니다.` 라는 메일이 도착해 있을 거예요.


{% endstep %}

{% step %}
### 링크 클릭하기

* 🔗 메일 본문을 보시면 중간에 \[ESG Reporting Tool] 이라는 링크가 보이실 겁니다. 그 링크를 '클릭'만 해주세요.
{% endstep %}
{% endstepper %}







## 공동작성 참여 인증

{% @arcade/embed flowId="a5UQDRgl53QSK7WiFGjS" url="https://app.arcade.software/share/a5UQDRgl53QSK7WiFGjS" %}



{% stepper %}
{% step %}
### 인증하기

* 인증하기 버튼을 눌러 공동작성 참여를 인증하세요.
* 인증 코드는 이미 자동으로 들어가 있습니다.
{% endstep %}

{% step %}
### 회원가입하기

보고서 작성에 앞서 회원가입이 필요합니다.

먼저 가입에 필요한 기본 정보를 입력해 주세요.

* 이메일 주소: 반드시 초대 메일을 받으신 이메일 주소를 그대로 입력해 주세요. (다른 메일로 가입할 경우, 배정된 보고서에 접근이 어려울 수 있습니다.)
* 이름 및 소속 조직명: 본인의 성함과 소속 조직명에 현재 재직 중인 회사명을 입력합니다.
* 비밀번호: 사용할 비밀번호를 입력합니다.&#x20;
  * 8자\~20자 이내
  * 영어 소문자, 숫자, 특수문자 조합
* 정보 입력을 모두 마쳤다면 하단의 \[가입하기] 버튼을 클릭합니다.


{% endstep %}

{% step %}
### 개인정보 제공 동의

개인정보 제공 동의를 완료해야 최종적으로 회원가입 및 서비스 이용이 가능합니다.
{% endstep %}
{% endstepper %}

## 보고서 수정이 불가능하신가요?

<figure><img src="../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

ERT 보고서는 효율적인 협업과 보안을 위해 파트별로 '접근 제한' 설정이 가능합니다. 만약 담당하신 파트임에도 불구하고 편집이나 삭제가 되지 않는다면, 해당 주제에 대한 권한이 제한되어 있을 수 있습니다.

이 경우, **보고서 소유자(관리자)**&#xC5D0;게 권한 변경을 요청하시거나 [QESG로 문의](https://odvvp.channel.io)해 주시면 빠르게 도와드리겠습니다.
