# 번역하기

### 자동 영문 번역 (국/영문 보고서 동시 산출)

QESG 플랫폼은 머신러닝 기반의 자동 영문 번역 기능을 통해 국문 보고서 작성 후 즉시 영문 보고서를 산출할 수 있도록 지원합니다. 이를 통해 글로벌 이해관계자 및 ESG 평가기관의 공시 요구에 신속하게 대응할 수 있습니다.

#### 1) 주요 특징 및 기술

* ESG 특화 머신러닝 번역: ESG 분야 언어로 사전 학습된 기계학습 API(ChatGPT, Google Translate API 등)를 활용하여 실시간 번역을 제공합니다.
* 서식 자동 유지 및 편집: 국문 텍스트에 적용된 서식이 영문 번역 시에도 자동으로 유지되며, 필요한 경우 영문 텍스트에만 독립적인 서식을 적용할 수도 있습니다.
* 수동 편집 지원: 자동 번역된 내용을 사용자가 직접 수정하거나 보완할 수 있는 편집 기능을 제공합니다.
* 고품질 번역 서비스(별도): 전문 번역사의 고품질 번역 및 감수가 필요한 경우 별도의 유료 서비스를 의뢰할 수 있습니다.

***

#### 2) 영문 번역 프로세스 (자동)

보고서의 '번역' 탭에서 간단한 설정만으로 전체 보고서를 번역할 수 있습니다.

1. 메뉴 접속: 상단의 \[번역] 탭을 선택합니다.
2. 영문명 설정: 보고서의 영문 명칭을 설정하고 \*\*\[저장하기]\*\*를 누릅니다.
3. 번역 실행: \[보고서 번역하기] 버튼을 클릭합니다. 전체 번역에는 약 15분 내외가 소요됩니다.
4. 완료 알림: 번역이 완료되면 화면 우측에 완료 알림 창이 표시됩니다.

> 참고: 이미 수동 번역을 진행한 지표가 있다면, 해당 지표를 제외한 나머지 지표에 대해서만 자동 번역이 수행됩니다.

***

#### 3) 번역 수정 및 관리 (수동)

자동 번역된 결과물을 사용자가 직접 검토하고 세밀하게 조정할 수 있습니다.

* 번역 수정: 화면 우측의 '영문' 영역에서 내용을 확인한 후, \*\*\[번역 수정하기]\*\*를 클릭하여 텍스트를 편집합니다.
* 저장 및 초기화: 수정 후 \*\*\[저장]\*\*을 누르면 반영되며, 다시 자동 번역본으로 되돌리고 싶을 때는 \[번역 초기화] 기능을 사용할 수 있습니다.
* 상태 표시: 수동 번역이 정상적으로 저장되면 해당 항목 하단에 체크($$ $\checkmark$ $$) 아이콘이 표시되어 작업 현황을 쉽게 파악할 수 있습니다.

***

#### 4) 번역 단계 생략 안내

영문 보고서 공시를 희망하지 않는 경우 번역 단계를 생략할 수 있습니다.

* 단, 이 경우 ‘디자인 테마 세부 설정’ 탭에서 \*\*\[언어전환 버튼 표시]\*\*를 비활성화해야 외부 이해관계자의 접근을 제한할 수 있습니다.
