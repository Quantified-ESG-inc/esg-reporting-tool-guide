# 번역하기

### 자동 영문 번역 (국/영문 보고서 동시 산출)

#### 주요 특징 및 기술

* 최첨단 AI 기반 번역: 구글의 최신 AI 모델인 Gemini 3를 도입하여, 단순 번역을 넘어 ESG 전문 용어와 복잡한 문맥을 정확하게 파악하는 고차원 실시간 번역을 제공합니다.
* 서식 자동 유지 및 편집: 국문 텍스트에 적용된 서식이 영문 번역 시에도 자동으로 유지되며, 필요한 경우 영문 텍스트에만 독립적인 서식을 적용할 수도 있습니다.
* 수동 편집 지원: 자동 번역된 내용을 사용자가 직접 수정하거나 보완할 수 있는 편집 기능을 제공합니다.
* 고품질 번역 서비스(별도): 전문 번역사의 고품질 번역 및 감수가 필요한 경우 별도의 유료 서비스를 의뢰할 수 있습니다. 이 경우 [QESG에 문의](https://odvvp.channel.io)해 주세요.

## 자동 영문 번역 프로세스

{% @arcade/embed flowId="Au0x7LtUgphbHKIBSPSZ" url="https://app.arcade.software/share/Au0x7LtUgphbHKIBSPSZ" %}



보고서의 '번역' 탭에서 간단한 설정만으로 전체 보고서를 번역할 수 있습니다.

1. 메뉴 접속: 상단의 \[번역] 탭을 선택합니다.
2. 영문명 설정: 보고서의 영문 명칭을 설정하고 \[저장하기]를 누릅니다.
3. 번역 실행: \[보고서 번역하기] 버튼을 클릭합니다. 전체 번역에는 약 15분 내외가 소요됩니다.
4. 완료 알림: 번역이 완료되면 화면 우측에 완료 알림 창이 표시됩니다.

> 참고: 이미 수동 번역을 진행한 지표가 있다면, 해당 지표를 제외한 나머지 지표에 대해서만 자동 번역이 수행됩니다.

## 수동 번역 수정 및 관리

{% @arcade/embed flowId="KEL8mMXLohQvqiQwYhre" url="https://app.arcade.software/share/KEL8mMXLohQvqiQwYhre" %}



자동 번역된 결과물을 사용자가 직접 검토하고 세밀하게 조정할 수 있습니다.

* 번역 수정: 화면 우측의 '영문' 영역에서 내용을 확인한 후, **\[번역 수정하기]**&#xB97C; 클릭하여 텍스트를 편집합니다.
* 저장 및 초기화: 수정 후 \*\*\[저장]\*\*을 누르면 반영되며, 다시 자동 번역본으로 되돌리고 싶을 때는 \[번역 초기화] 기능을 사용할 수 있습니다.
* 상태 표시: 수동 번역이 정상적으로 저장되면 해당 항목 하단에 체크($$ $\checkmark$ $$) 아이콘이 표시되어 작업 현황을 쉽게 파악할 수 있습니다.

## 번역 버튼 비활성화 하기

<mark style="color:$info;">국문 보고서만 발간하고 싶으신가요? 영문 보고서 공시를 희망하지 않는 경우 번역 단계를 생략할 수 있습니다. 단, 이 경우 ‘디자인 테마 세부 설정’ 탭에서 \[언어전환 버튼 표시]를 비활성화해야 외부 이해관계자의 접근을 제한할 수 있습니다.</mark>

{% @arcade/embed flowId="6rbnBxqI03VAnAqVDjQX" url="https://app.arcade.software/share/6rbnBxqI03VAnAqVDjQX" %}

*   보고서의 디자인과 사용자 경험을 최적화하기 위해 불필요한 번역 버튼 노출을 제어할 수 있습니다.

    * 기본 설정 접속: 프로젝트 관리 화면에서 해당 보고서를 클릭해 \[기본 설정] 메뉴를 선택하여 진입합니다.
    * 디자인 테마 설정 이동: 좌측 사이드바 메뉴에서 \[디자인 테마 설정] 항목을 클릭합니다.
    * 세부 설정 페이지 진입: 디자인 테마 목록 중 현재 적용된 테마의 \[디자인 테마 세부 설정] 버튼을 눌러 상세 설정 화면으로 이동합니다.
    * 번역 기능 비활성화: 설정 항목 리스트에서 \[언어 전환 버튼] 스위치를 찾아 \[비활성화(OFF)] 상태로 변경한 뒤 저장합니다.

    이와 같이 설정하면 보고서 뷰어 화면에서 번역 선택 버튼이 더 이상 노출되지 않습니다.
