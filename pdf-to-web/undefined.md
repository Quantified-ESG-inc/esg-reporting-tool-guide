# \[도입] 디지털 전환 시작하기

## ESG 공시, 왜 디지털로 전환해야 하나요?

대부분의 기업이 ESG 보고서라고 하면 두꺼운 책자나 수백 페이지의 PDF 파일을 먼저 떠올립니다. 하지만 처음 시작하는 기업일수록 전통적인 PDF 방식은 실무자에게 큰 부담이 됩니다.

⚠️ 기존 PDF 보고서 발간의 한계

* 자원 과다 투입: 매년 발간할 때마다 새로운 표지와 내지 디자인을 개발해야 하므로 디자인 의사결정에 많은 시간과 비용이 소모됩니다.
* 수정의 어려움: 발간 후 오탈자가 발견되어도 이미 배포된 PDF 파일을 수정하거나 다시 배포하기가 매우 어렵고 시간이 오래 걸립니다.
* 연속성 결여: 작년에 만든 보고서를 활용하기 어렵고, 매년 처음부터 다시 디자인 외주를 맡겨야 하는 반복적인 비용이 발생합니다.

## 기존 PDF 보고서의 한계를 넘어 Web 보고서로 전환하세요

디지털로 전환된 웹 보고서는 단순히 '보기 편한 것'을 넘어, 기업의 ESG 데이터를 자산화하고 확산시키는 강력한 도구가 됩니다.

#### 유지·보수의 편의성 및 실시간 대응

발간 이후 발생하는 오류나 업데이트 상황에 유연하게 대처할 수 있습니다.

* 오탈자 직접 수정: 전문 디자인 업체에 수정을 요청하고 기다릴 필요 없이, 웹상에서 담당자가 직접 오탈자를 즉시 수정할 수 있습니다.
* 상시 피드백 반영: 이해관계자의 수정 요구나 업데이트 사항을 실시간으로 반영하여 공시 정보의 정확도를 최상으로 유지합니다.

#### &#x20;검색 엔진 최적화(SEO) 및 정보 접근성 확대

PDF는 파일을 다운로드해야만 내용을 볼 수 있지만, 웹 보고서는 정보의 확산성 차원이 다릅니다.

* 서치 엔진 검색 가능: 구글 등 주요 검색 엔진에서 보고서 내의 구체적인 키워드가 직접 검색되어 이해관계자와의 접점이 넓어집니다.
* 주제별 고유 URL 부여: '기후변화 대응' 등 특정 섹션마다 고유한 주소를 부여할 수 있어, 평가기관이나 투자자에게 필요한 핵심 정보를 즉시 연결해 줄 수 있습니다.



#### 차년도 보고의 연속성 및 비용 절감 (Cloning)

매년 새로운 디자인을 위해 막대한 비용과 시간을 들일 필요가 없습니다.

* 보고서 복제 기능: 전년도 보고서의 레이아웃과 데이터를 그대로 복제(Clone)하여 올해의 수치만 업데이트하는 방식으로 작업할 수 있습니다.
* [▶ 보고서  복제하여 발간해보기](https://www.google.com/search?q=%23)
* 디자인 자산화: 메인 페이지 디자인을 유지하며 내용만 수정하므로, 매년 반복되던 디자인 외주 비용과 의사결정 시간을 획기적으로 줄여줍니다.
* [▶ 내용   수정해보기](https://www.google.com/search?q=%23)





#### ESG Reporting Tool (ERT)란 무엇인가요?

ERT(ESG Reporting Tool)는 기업의 복잡한 ESG 데이터를 체계적으로 관리하고, 글로벌 공시 기준에 부합하는 보고서를 신속하게 생성할 수 있도록 지원하는 ESG 공시 전문 솔루션입니다.



## 웹 보고서, 왜 ERT여야 할까요?

ERT는 숙련된 담당자가 소수의 인력으로도 보고서의 퀄리티를 높이면서 발간 소요 기간을 획기적으로 단축할 수 있게 돕습니다.

#### 1. AI 기반 PDF to Web자동 변환&#x20;

디지털 전환을 망설이는 가장 큰 이유는 '기존 데이터의 재구축 부담'입니다. ERT는 이를 AI 기술로 해결했습니다.

* 손쉬운 초기 구축: 기존에 발간했던 PDF 파일을 업로드하기만 하면, AI가 이를 분석하여 HTML 웹 형태로 자동 변환합니다.
* 리소스 절감: 처음부터 웹 페이지를 새로 디자인하고 텍스트를 옮길 필요 없이, 기존의 보고 내용을 그대로 활용하여 아주 쉽고 빠르게 디지털 공시 시스템을 구축할 수 있습니다.
* [▶ PDF 변환해보기](https://www.google.com/search?q=%23)

#### 2. 텍스트가 읽히는 HTML 기반 도식&#x20;

일반적인 웹 보고서가 도식이나 조직도를 '이미지'로 처리하는 것과 달리, ERT는 모든 콘텐츠를 HTML 코딩 기반으로 구현합니다.

* 검색 및 수집 가능(Crawlable): 도식 내의 모든 텍스트가 이미지 데이터가 아닌 실제 텍스트 데이터로 인식됩니다.
* 평가 점수 상향: 최근 ESG 평가 기관들은 데이터를 자동으로 크롤링하여 분석합니다. 이미지는 읽지 못해 누락될 수 있는 성과들이 ERT에서는 모두 검색 및 읽기가 가능하여 평가 대응에 압도적으로 유리합니다.

#### 3. 엑셀 연동 정량 데이터 자동화&#x20;

정량 데이터를 사람이 직접 입력하는 방식은 오류를 피하기 어렵습니다. ERT는 시스템적 자동화를 통해 신뢰성을 확보합니다.

* Excel to Web 변환: 엑셀에 정리된 정량 데이터를 업로드하면 즉시 웹 보고서의 수치와 차트로 구현됩니다.
* 오류 Zero & 검수 기능: 수기 입력 시 발생하는 휴먼 에러를 원천 차단하며, 오히려 기존 데이터에 포함되어 있던 논리적 오류까지 시스템이 자동으로 검수하여 보고서의 신뢰도를 높입니다.
* [▶ Excel 변환해보기](https://www.google.com/search?q=%23)

#### 4. 웹과 PDF의 동시 아카이빙

웹 공시는 실시간 수정이 장점이지만, 자칫 과거 기록이 덮어씌워져 연도별 비교가 어려워질 수 있습니다. ERT는 이를 이원화된 시스템으로 관리합니다.

* 웹아카이빙: 최신 성과는 인터랙티브한 웹으로 보여주면서도,&#x20;
* PDF 아카이빙:  연도별 확정본은 PDF 형태로 함께 보관합니다. 전년도 보고 내용을 언제든 대조·확인할 수 있어, 기업의 ESG 성과 히스토리를 체계적으로 관리하고 비교 분석하기에 용이합니다.

#### 5. 전사적 협업 플랫폼&#x20;

보고서 발간은 여러 부서의 협업이 필수적입니다. ERT는 이 과정을 이메일이나 메신저가 아닌 시스템 내로 통합했습니다.

* 직접 참여 및 코멘트: 실무에 참여하는 타 부서 인원들이 직접 ERT에 접속하여 원고를 수정하거나 특정 지표에 코멘트를 달 수 있습니다.
* 커뮤니케이션 최적화: 자료 취합부터 피드백 반영까지 모든 과정이 기록되어, 보고서 발간 기간 동안 발생하는 불필요한 소통 비용을 획기적으로 줄여줍니다.
