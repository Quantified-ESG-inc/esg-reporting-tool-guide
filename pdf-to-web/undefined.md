# \[도입] 디지털 전환 시작하기

## ESG 공시, 왜 디지털로 전환해야 하나요?

대부분의 기업이 ESG 보고서라고 하면 두꺼운 책자나 수백 페이지의 PDF 파일을 먼저 떠올립니다. 하지만 처음 시작하는 기업일수록 전통적인 PDF 방식은 실무자에게 큰 부담이 됩니다.

⚠️ 기존 PDF 보고서 발간의 한계

* 자원 과다 투입: 매년 발간할 때마다 새로운 표지와 내지 디자인을 개발해야 하므로 디자인 의사결정에 많은 시간과 비용이 소모됩니다.
* 수정의 어려움: 발간 후 오탈자가 발견되어도 이미 배포된 PDF 파일을 수정하거나 다시 배포하기가 매우 어렵고 시간이 오래 걸립니다.
* 연속성 결여: 작년에 만든 보고서를 활용하기 어렵고, 매년 처음부터 다시 디자인 외주를 맡겨야 하는 반복적인 비용이 발생합니다.

####





## 기존 PDF 보고서의 한계를 넘어 Web 보고서로 전환하세요

디지털로 전환된 웹 보고서는 단순히 '보기 편한 것'을 넘어, 기업의 ESG 데이터를 자산화하고 확산시키는 강력한 도구가 됩니다.









#### ESG Reporting Tool (ERT)란 무엇인가요?

\*\*ERT(ESG Reporting Tool)\*\*는 기업의 복잡한 ESG 데이터를 체계적으로 관리하고, 글로벌 공시 기준에 부합하는 보고서를 신속하게 생성할 수 있도록 지원하는 ESG 공시 전문 솔루션입니다.
