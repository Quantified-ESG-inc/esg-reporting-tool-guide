# \[4단계] PDF 보고서 Web으로 변환하기

이미 완성된 PDF 보고서가 있다면, 처음부터 모든 텍스트를 다시 입력할 필요가 없습니다. 기존 PDF 디자인을 캡처(이미지화)하거나 PDF 파일을 첨부하여 AI에 요청하는 것만으로도 훌륭한 디지털 보고서가 완성됩니다.

{% stepper %}
{% step %}
#### 1단계. 자료 업로드 및 AI 도우미  열기

* 기존 PDF 보고서에서 웹으로 옮기고 싶은 핵심 페이지나 디자인 요소를 PDF 파일 또는 이미지 파일(`PNG`, `JPG`)로  준비합니다.
  * 📄PDF 파일 Tip: 10MB 이하의 파일만 업로드가 가능하므로 파일을 최대 3\~4페이지로 분할하여 올려주세요.
  * 📷이미지캡쳐 Tip: 웹에서 글자가 깨지지 않도록 가급적 고해상도 상태에서 캡처해 주세요.
{% endstep %}

{% step %}
#### 2단계. 분석 및 초안 생성 요청하기

* 요청사항 입력: 첨부한 파일에 대해 AI가 수행할 작업을 입력합니다.
  * _예: "이 PDF의 내용을 보고서 본문에 넣어줘. 디자인도 비슷하게 유지해줘."_
* AI 생성: 입력 후 약 1분 내외의 시간이 지나면 AI가 자료를 분석하여 보고서 초안을 제안합니다.
{% endstep %}

{% step %}
#### 3단계. 본문에 반영하기

* 본문에 추가: AI가 생성한 결과물을 확인하고, \[본문에 추가] 버튼을 눌러 즉시 보고서 섹션에 반영합니다.
* PDF의 대제목, 중제목까지 모두 그대로 반영한 웹 보고서 초안이 생성된 것을 확인하실 수 있습니다.
* 대제목, 중제목을 기반으로 **왼쪽에 네비게이션바가 자동으로 생성**됩니다.


{% endstep %}
{% endstepper %}





### AI로 작성한 내용 재수정하기

AI로 변환한 웹 보고서 초안은 AI 재작성 기능을 활용하여 수정할 수 있습니다.

작년의 PDF 내용을 기반으로 올해 내용을 쉽게 작성해 보세요.

{% embed url="https://app.arcade.software/share/BNrxatLBSKBaxLDAXYps" %}



{% stepper %}
{% step %}
#### 1단계. AI 재작성 요청하기

* 아이콘 클릭: AI 도우미가 답변을 생성한 후, 결과물 하단에 있는 \[AI 재작성] 버튼을 클릭합니다.
* 수정 모드 진입: 이 버튼을 누르면 AI가 기존에 작성한 내용을 바탕으로 새로운 요청을 받을 준비를 합니다.
{% endstep %}

{% step %}
#### 2단계. 상세 수정 및 디자인 요청 사항 입력하기

* 구체적 요청: 입력창에 원하는 수정 방향을 자유롭게 입력해 주세요.
  * 행 추가: "올해 성과를 넣을 수 있도록 데이터 테이블의 열을 하나 추가해줘."
  * 내용 수정: "이 내용을 조금 더 요약해줘", "전문적인 용어를 사용해서 다시 써줘"
  * 디자인 변경: "이 데이터를 읽기 편하게 표(Table) 형태로 그려줘", "핵심 내용을 불렛 포인트로 정리해줘"
* 전송: 요청 사항을 입력한 후 전송 아이콘을 누르면 AI가 즉시 수정을 시작합니다.
{% endstep %}

{% step %}
#### 3단계. 수정 결과 확인 및 본문 반영하기

* 결과 비교: AI가 새롭게 제안한 결과물을 확인합니다.
* 본문에 추가: 수정된 내용이 만족스럽다면 \[이걸로 본문 수정] 버튼을 클릭하세요. 기존 본문 내용이 새롭게 다듬어진 내용으로 즉시 교체됩니다.
{% endstep %}
{% endstepper %}





#### 💡 작성 팁: AI 재작성 200% 활용법

* 디자인 수정: 텍스트 위주의 복잡한 수치 데이터를 " 색감을 환경에 맞게 수정해줘 " 또는 "도식으로 깔끔하게 정리해줘 "라고 요청해 보세요. 보고서의 디자인 뿐만 아니라 시각적 가독성과 전문성이 획기적으로 높아집니다.
* 톤앤매너 고도화: "우리 회사의 브랜드 가이드에 맞게 친절하면서도 신뢰감 있는 말투로 다듬어줘"와 같이 구체적인 스타일을 지정할수록 더 만족스러운 결과를 얻을 수 있습니다.
* 지속적인 피드백: 한 번에 완벽하지 않아도 괜찮습니다. 재작성 기능을 반복 활용하여 원고의 완성도를 차근차근 높여보세요.
