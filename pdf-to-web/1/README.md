# \[1단계] 새 보고서 생성하기

{% hint style="info" %}
[회원가입](../../ert/undefined.md), [결제](../../ert/undefined-1.md)를 마친 이후 새 보고서를 생성할 수 있습니다.&#x20;
{% endhint %}



## 새 보고서 생성하기

보고서를 발행하는 주체(기업/기관)의 정보를 등록합니다.

* 검색하여 입력: 검색창에 기관명을 입력한 후 돋보기 아이콘을 눌러 선택하면 기관명, 본사 주소, CI/로고가 자동 입력됩니다.
* 직접 입력: 검색 결과에 기관이 없을 경우, 기관명과 주소 등 관련 정보를 직접 입력하고 로고 파일을 업로드할 수 있습니다.

#### 보고 형태 및 프레임워크 선택

작성하고자 하는 보고서의 목적에 맞는 형식과 국제 표준을 선택합니다.

* 보고 형태 선택:
  * Web Report(추천): 기존 PDF의 풍부한 내용을 디지털로 완벽하게 이식합니다. 표지, 메뉴, 개별 페이지로 구성된 독자적인 웹사이트 형태로, 이해관계자에게 최고의 접근성을 제공합니다.
  * Index Table: GRI, SASB 등 국제 표준 기반의 참조 테이블 위주로 구성된 간단한 보고서입니다.
* 보고 프레임워크 및 부가 기능:
  * 표준 선택: GRI Standards, SASB Standards, TCFD Framework 중 필요한 표준을 선택합니다.
  * 추가 옵션: 필요에 따라 Data Center 등 부가 기능을 함께 선택할 수 있습니다.
* 생성 완료: 모든 설정을 마친 후 \*\*\[보고서 생성하기]\*\*를 누르면 대시보드에 접근 버튼이 생성됩니다.

***

#### ⚙️ 설정한 정보를 수정하고 싶다면?

보고서 생성 시 입력한 기본 정보는 언제든지 수정이 가능합니다.

1. 보고서 상단 메뉴 중 \[기본설정] 탭으로 이동합니다.
2. 좌측 메뉴에서 \*\*\[보고서 설정] > \[기본 정보]\*\*를 차례로 클릭합니다.
3. 화면 하단의 \[기본정보 수정하기] 버튼을 눌러 보고서 명, 기간, 기관 정보 등을 업데이트할 수 있습니다.

> 확인 가능한 정보: 해당 메뉴에서는 최초 생성일, 최신 업데이트 일자, 소유자 및 현재 진행 상태도 함께 확인 가능합니다.



## 보고서 복제 및 활용 가이드

#### 내년에는 올해 보고서를 활용해 더욱 손쉽게 작성하세요.

💡 내년 보고서 작성, 올해보다 훨씬 쉬워집니다!

* 보고서 복제 기능: 올해 정성껏 구축한 보고서의 틀(목차, 디자인, 기본 데이터)을 내년에도 그대로 활용하실 수 있습니다. '보고서 복제' 기능을 사용하면 매년 처음부터 시작할 필요 없이, 변경된 데이터와 성과만 업데이트하여 신속하게 차기년도 보고서를 발행할 수 있습니다.
* 지속적인 데이터 관리: 과거 보고서 데이터를 기반으로 시계열 분석을 수행하거나, 전년도 대비 개선된 성과를 시각화하는 데 매우 유리합니다.
