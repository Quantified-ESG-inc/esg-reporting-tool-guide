# 차년도 보고에 활용하기





## 보고서 복제하기

{% stepper %}
{% step %}
#### 1단계. 기존 보고서 메뉴 진입하기

* 보고서 선택: 작년에 작성 완료된 보고서 중, 내년도 보고서의 기초로 사용할 보고서를 찾습니다.
* 더보기 클릭: 해당 보고서 카드나 리스트에 있는 \[더보기(...)] 아이콘을 클릭하여 관리 메뉴를 엽니다.


{% endstep %}

{% step %}
#### 2단계. 보고서 복제 실행하기

* 복제 기능 선택: 펼쳐진 메뉴에서 \[복제] 버튼을 선택합니다.
* 자동 생성: 클릭 한 번으로 기존 보고서의 레이아웃과 데이터가 그대로 담긴 새로운 보고서가 빠르게 생성됩니다.


{% endstep %}

{% step %}
#### 3단계. 올해 보고서 정보로 수정하기

* 기본 정보 변경: 복제된 보고서의 상세 화면으로 이동하여, 보고서 제목의 연도나 프로젝트 기간 등 기본 정보를 내년에 맞게 수정합니다.
* 내용 업데이트: 기존에 입력되어 있던 작년 데이터를 기반으로, 변경된 부분만 수정하거나 새로운 데이터를 입력하여 보고서를 완성합니다.
{% endstep %}
{% endstepper %}
