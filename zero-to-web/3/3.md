# 방법3: 샘플 불러오기

{% embed url="https://app.arcade.software/share/egJpuj5wV98kCjgIAtAr" %}





{% stepper %}
{% step %}
#### 1단계. 샘플 불러오기 시작하기

* 방식 선택: 목차 작성 방식 선택 화면에서 세 번째 항목인 \[샘플 불러오기] 를 클릭합니다.
{% endstep %}

{% step %}
#### 2단계. 활용할 샘플 보고서 선택하기

* 보고서 선택: 화면에 나타난 샘플 목록 중 우리 기업의 보고서 작성 목적에 가장 부합하는 사례를 선택합니다.
* 샘플 로드: 선택 시 해당 보고서에 구성되어 있던 표준 대분류와 소분류 목차가 편집 화면에 자동으로 불러와집니다.
{% endstep %}

{% step %}
#### 3단계. 목차 검토 및 최종 저장하기

* 내용 검토: 불러온 샘플 목차의 구성(ERT 저장 기본 목차 등)이 우리 기업의 방향성과 맞는지 확인합니다.
* 수정 및 보완: 샘플 내용을 바탕으로 문구를 수정하거나, \[+] 버튼을 눌러 우리 기업만의 고유한 목차 항목을 추가합니다.
* 최종 저장: 목차 구성이 완료되었다면 좌측 하단의 \[목차 저장하기] 버튼을 클릭하여 모든 설정을 완료합니다.
{% endstep %}
{% endstepper %}

***

#### 💡 샘플  불러오기 작성 팁

* 검증된 구조: 샘플 목차는 공시 기준을 충족하는 우수 사례를 바탕으로 합니다. 구조 잡기가 막막할 때 가이드라인으로 활용해 보세요.
* 산업 특화 지표 추가: 표준 목차는 보편적인 가이드를 제공하므로, 우리 업종만의 특수한 공시 항목이 누락되지 않았는지 점검하고 추가하세요.
* 자유로운 편집: 샘플은 말 그대로 '본보기'일 뿐입니다. 불러온 후에는 자유롭게 삭제하거나 순서를 바꿀 수 있으니 부담 없이 시작해 보세요.
