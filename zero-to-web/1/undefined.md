# 메인 페이지 제작하기

### AI로 메인 페이지 디자인하기

보고서의 얼굴인 메인 페이지를 어떻게 꾸며야 할지 고민되시나요? AI 도우미가 보고서의 주제와 기업 정보를 분석하여 최적의 메인 디자인을 제안해 드립니다.

{% embed url="https://app.arcade.software/share/ey8WqQYVA0PzEuXhYumi" %}







***

#### ① 1단계. 메인 페이지 편집 모드 진입하기

* 보고서 선택: 편집하고자 하는 보고서를 클릭하여 상세 화면으로 진입합니다.
* 탭 이동: 상단 메뉴 탭 중에서 \[메인페이지] 탭을 클릭합니다. 이곳에서 메인 페이지의 전체적인 디자인과 내용을 확인하거나 수정할 수 있습니다.

#### ② 2단계. AI를 통한 자동 디자인 생성하기

* AI 생성 요청: 메인 페이지 탭 내에서 AI 디자인 생성 기능을 실행합니다.
* 자동 구성: AI가 보고서의 성격에 맞는 배경 이미지, 타이틀 문구, 레이아웃을 자동으로 구성합니다. 클릭 몇 번만으로 전문가가 디자인한 것 같은 결과물을 얻을 수 있습니다.

#### ③ 3단계. 콘텐츠 확인 및 맞춤 편집하기

* 내용 검토: AI가 생성한 타이틀, 부제목, 핵심 문구 등이 우리 기업의 방향성과 맞는지 확인합니다.
* 직접 수정: 생성된 디자인 위에 우리 기업의 로고를 추가하거나, 특정 문구를 직접 수정하여 완성도를 높여보세요.

***

#### 💡 작성 팁: 매력적인 메인 페이지를 위한 노하우

* 핵심 키워드 강조: 메인 페이지 타이틀에는 이번 보고서의 핵심 테마(예: '탄소중립', '신뢰경영' 등)를 명확하게 노출하는 것이 좋습니다.
* 시각적 일관성: AI가 제안한 배경색이나 이미지가 우리 기업의 브랜드 가이드라인과 일치하는지 확인해 보세요. 필요하다면 AI 재작성 기능을 통해 분위기를 조정할 수 있습니다.
* 모바일/웹 최적화 확인: 메인 페이지는 웹 보고서의 첫 화면인 만큼, 다양한 기기에서 어떻게 보이는지 미리보기를 통해 점검해 보세요.







### AI로 메인 페이지 수정하기

이미 생성된 메인 페이지의 디자인이나 텍스트가 조금 아쉽다면, AI에게 구체적인 요구사항을 전달하여 실시간으로 수정해 보세요.

{% embed url="https://app.arcade.software/share/9dMCGxSKUyRdbWRm6xYZ" %}



#### 원하는 부분 선택 및 수정 요청하기

* 수정 모드 진입: 상단 \[메인페이지] 탭에서 수정하고 싶은 영역(배경, 타이틀, 요약 문구 등)을 확인합니다.
* AI 도우미 활용: 우측 상단의 AI 도우미 아이콘을 클릭하거나, 특정 영역의 수정 버튼을 통해 AI에게 변경 사항을 요청합니다.
  * _디자인 변경 예시: "배경 이미지를 조금 더 밝고 희망찬 느낌의 자연 풍경으로 바꿔줘."_
  * _텍스트 수정 예시: "타이틀 문구를 우리 회사의 올해 슬로건인 '함께 만드는 미래'를 포함해서 더 임팩트 있게 고쳐줘."_

#### AI 제안 확인 및 실시간 반영

* 디자인 업데이트: AI가 요청 사항을 반영하여 새로운 디자인 시안이나 문구를 제안합니다.
* 즉시 적용: 제안된 내용이 마음에 든다면 \[반영하기] 버튼을 클릭하세요. 메인 페이지에 즉시 적용되어 변화된 모습을 확인할 수 있습니다.
* 반복 수정: 결과가 만족스러울 때까지 "조금 더 심플하게", "글자색을 파란색 계열로" 등 추가 요청을 통해 완성도를 높일 수 있습니다.

***

#### 💡 메인 페이지 수정 팁

* 이미지 퀄리티 높이기: AI에게 배경 이미지를 요청할 때 "고화질", "미니멀한", "기업 친화적인" 등의 형용사를 함께 사용하면 더 세련된 결과를 얻을 수 있습니다.
* 브랜드 컬러 유지: 우리 기업의 상징 색상이 있다면 "로고의 메인 컬러인 녹색이 강조되도록 디자인해줘"와 같이 요청하여 브랜드 일관성을 지켜보세요.
* 가독성 체크: 배경 이미지 위에 글자가 잘 보이지 않는다면, AI에게 "글자가 더 잘 보이도록 배경에 어두운 필터를 씌워줘"라고 요청해 보세요.
