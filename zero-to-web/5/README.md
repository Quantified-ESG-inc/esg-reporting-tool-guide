# \[5단계] 검증, 번역 및 웹 보고서 발간

보고서의 본문 작성을 마쳤다면, 이제 제3자 검증, 번역, 공개 단계를 통해 대외 신뢰도를 확보하고 전 세계 이해관계자에게 결과물을 공유해 보세요.

&#x20;

***



## 1. 제3자 검증: 보고서의 무게를 더하기

단순히 정보를 나열하는 것을 넘어, 공시 데이터의 객관성과 신뢰도를 증명하는 과정입니다.

* 검증의 필요성: 투자자 및 평가기관은 공시 데이터의 정확성을 확인하기 위해 제3자 검증을 필수로 요구하는 추세입니다.
* 진행 방식: 외부 전문 기관을 통하거나, ERT(QESG)가 제공하는 독립적인 검증 툴을 선택해 효율적으로 진행할 수 있습니다.
* 참고 사항: 검증은 필수 조건은 아니지만, 대외 신뢰 확보를 위해 권장되는 절차이며 검증 완료 후 검증의견서를 수령하게 됩니다.
* 검증에 대한 자세한 설명은 [검증하기 바로가기](undefined.md)를 참고해 주세요.





## 2. 자동 영문 번역: 글로벌 공시 대응

국문 보고서가 완성되면 별도의 번역 업체 없이도 즉시 글로벌 버전을 산출할 수 있습니다.

* AI 기반 번역: ESG 분야 언어에 사전 학습된 기계학습 API(ChatGPT 등)를 활용해 실시간으로 고품질 번역을 제공합니다.
* 서식 유지: 국문 텍스트에 적용된 모든 디자인 서식과 레이아웃이 영문 버전에서도 그대로 유지되어 편집 시간을 단축해 줍니다.
* 수동 편집 가능: 자동 번역된 결과물을 사용자가 직접 검토하고 세부 문구(Wording)를 수정하거나 보완할 수 있습니다.
*   번역에 대한 자세한 설명은 [번역하기 바로가기](undefined-1.md)를 참고해 주세요.

    &#x20;

## 3. 최종 발간: 웹과 PDF로 즉시 공개

검증과 번역이 완료된 보고서를 클릭 한 번으로 세상에 공개하는 단계입니다.

* 웹 보고서 공개: 고유한 웹 링크(URL) 형태로 발간되어 누구나 쉽게 접속할 수 있으며, 주소(도메인)를 직접 수정할 수도 있습니다.
* PDF 산출: 인쇄용이나 아카이빙을 위한 PDF 파일 형태의 보고서도 동시에 생성 가능합니다.
* 공개 범위 설정: 수정이 필요하거나 특정 시점에 공개해야 한다면 '비공개 전환' 기능을 통해 접근을 제한할 수 있습니다.
*   최종 발간에 대한 자세한 설명은 [최종 보고서 발간하기 (웹, PDF)](pdf.md)를 참고해 주세요.
