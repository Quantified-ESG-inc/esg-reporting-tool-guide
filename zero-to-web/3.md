# \[3단계] 목차 작성하기



#### 목차 작성을 시작하기 전에: 중요성 평가 실시하기

본격적인 목차 작성에 앞서, 우리 기업에 가장 중요한 ESG 이슈를 선정하는 '중요성 평가'가 선행되어야 합니다. 평가 결과를 바탕으로 보고서의 핵심 주제를 결정해 보세요.

* ERT 내 실시 불가: 현재 ERT(ESG Reporting Tool) 서비스 내에서는 자체적인 중요성 평가 기능을 제공하지 않습니다.
* 전문가 지원 요청: 중요성 평가 프로세스(설문조사, 분석 등)가 막막하시다면 QESG에 별도 문의해 주세요. 별도 프로젝트 및 과업을 통해 전문적인 이슈 도출을 지원해 드립니다.

👉 \[QESG에 중요성 평가 지원 [문의하기](https://odvvp.channel.io)]

> Tip: 중요 주제가 이미 선정되었다면, 아래의 \[2단계] 목차 구성 가이드를 따라 보고서의 뼈대를 완성해 보세요.

우리 기업의 준비 상황에 맞춰 아래 3가지 방법 중 하나를 선택하여 시작할 수 있습니다.

#### 💡 작성 팁

어떤 방법으로 시작하더라도 생성된 이후에는 자유롭게 대분류와 소분류를 추가, 삭제하거나 순서를 변경할 수 있습니다. 우리 기업의 중요성 평가 결과가 가장 잘 드러날 수 있도록 목차를 최적화해 보세요.





### 방법1 : 새 목차 생성하기

{% @arcade/embed flowId="sVSVa88p26o5IOusoz2v" url="https://app.arcade.software/share/sVSVa88p26o5IOusoz2v" %}



{% stepper %}

### 1단계. 새 목차 생성 시작하기

* 보고서를 선택해 편집 화면으로 이동합니다.&#x20;
* 목차 작성 방식 선택 화면에서 첫 번째 항목인 \[새 목차 생성]을 클릭합니다.


{% endstep %}

{% step %}
### 2단계. 대분류 및 소분류 구성하기

편집 화면에 진입하면 본격적으로 보고서의 뼈대를 잡을 수 있습니다.

* 목차 추가하기: 화면 중앙의 \[+ 목차 추가하기] 버튼을 클릭하여 첫 번째 목차 항목을 생성합니다.
* 항목 입력: \* 대분류: 보고서의 큰 장(Chapter) 제목을 입력합니다. (예: 환경 경영, 사회적 책임 등)
* 소분류: 해당 장에 포함될 세부 항목을 입력합니다. (예: 온실가스 배출량 관리, 안전보건 정책 등)
* 소분류 추가: 소분류 입력창 아래의 \[+] 버튼을 누르면 해당 대분류 안에 더 많은 소분류 항목을 계속해서 추가할 수 있습니다.
{% endstep %}

{% step %}
### 3단계. 목차 확장 및 저장하기

* 대분류 추가: 하나의 대분류 작성이 끝났다면, 하단의 \[+ 목차 추가하기] 버튼을 다시 눌러 다음 대분류를 생성합니다.
* 목차 저장: 모든 목차 구성이 완료되었다면 화면 좌측 하단의 \[목차 저장하기] 버튼을 클릭하여 설정을 완료합니다.
{% endstep %}
{% endstepper %}

#### 💡 새 목차 생성하기 작성 팁

이 방법은 처음부터 직접 구조를 설계해야 하므로 '논리적 설계'가 가장 중요합니다.

* 전체 스토리라인 먼저 구상하기: 빈 페이지에서 시작하는 만큼, 작성 전 보고서의 핵심 테마(예: '친환경 혁신' 등)를 먼저 정하면 대분류 간의 흐름을 잡기가 훨씬 수월해집니다.
* 일관된 명칭 사용: 대분류와 소분류의 명칭을 'OO 관리', 'OO 대응' 처럼 일관된 종결 어미로 맞추면 보고서의 전문성이 높아집니다.









### 방법2: 산업별 목차 불러오기

{% @arcade/embed flowId="HFNLKvEweqn2rQu0NNo8" url="https://app.arcade.software/share/HFNLKvEweqn2rQu0NNo8" %}

{% stepper %}
{% step %}
#### 1단계. 산업별 목차 선택하기

* 보고서를 선택해 편집 화면으로 이동합니다.
* 목차 작성 방식 선택 화면에서 두 번째 항목인 \[산업별 목차 불러오기]를 클릭합니다.
{% endstep %}

{% step %}
#### &#x20;2단계. 우리 기업의 업종 설정하기

* 업종 선택: 드롭다운 메뉴를 클릭하여 우리 기업의 산업군(제조, IT, 유통, 금융 등)을 선택합니다.
* 불러오기 실행: 업종을 선택한 후 \[선택 완료] 또는 \[목차 불러오기] 버튼을 클릭합니다.
{% endstep %}

{% step %}
#### 3단계. 생성된 목차 확인 및 편집하기

* 자동 구성 확인: 선택한 업종의 특성이 반영된 대분류와 소분류 목차가 화면에 자동으로 펼쳐집니다.
* 커스텀 편집: 자동 생성된 목차를 바탕으로 우리 기업의 상황에 맞춰 내용을 수정하거나 항목을 추가할 수 있습니다.
* 최종 저장: 구성이 완료되었다면 좌측 하단의 \[목차 저장하기] 버튼을 눌러 설정을 완료합니다.
{% endstep %}
{% endstepper %}

#### 💡 산업별 목차 불러오기 작성 팁

이 방법은 목차를 '우리 기업의 산업에 맞게 최적화'하는 것이 핵심입니다.

* 중요성 평가 결과 반영: 자동 생성된 목차 중 우리 기업의 '중요성 평가(Materiality Assessment)'에서 핵심 이슈로 꼽힌 항목이 누락되지 않도록 확인하세요.
* 표준 가이드라인 활용: 제공된 목차는 글로벌 공시 기준(GRI, SASB 등)을 준수하고 있습니다. 가급적 기본 틀을 유지하면서 우리 기업의 데이터를 매칭하는 것이 공시 신뢰도를 높이는 방법입니다.







### 방법3: 샘플 불러오기

{% @arcade/embed flowId="egJpuj5wV98kCjgIAtAr" url="https://app.arcade.software/share/egJpuj5wV98kCjgIAtAr" %}

{% stepper %}
{% step %}
#### 1단계. 샘플 불러오기 시작하기

* 보고서를 선택해 편집 화면으로 이동합니다.
* 목차 작성 방식 선택 화면에서 세 번째 항목인 \[샘플 불러오기]를 클릭합니다.
{% endstep %}

{% step %}
#### 2단계. 활용할 샘플 보고서 선택하기

* 보고서 선택: 화면에 나타난 샘플 목록 중 우리 기업의 보고서 작성 목적에 가장 부합하는 사례를 선택합니다.
* 샘플 로드: 선택 시 해당 보고서에 구성되어 있던 표준 대분류와 소분류 목차가 편집 화면에 자동으로 불러와집니다.
{% endstep %}

{% step %}
#### 3단계. 목차 검토 및 최종 저장하기

* 내용 검토: 불러온 샘플 목차의 구성(ERT 저장 기본 목차 등)이 우리 기업의 방향성과 맞는지 확인합니다.
* 수정 및 보완: 샘플 내용을 바탕으로 문구를 수정하거나, \[+] 버튼을 눌러 우리 기업만의 고유한 목차 항목을 추가합니다.
* 최종 저장: 목차 구성이 완료되었다면 좌측 하단의 \[목차 저장하기] 버튼을 클릭하여 모든 설정을 완료합니다.
{% endstep %}
{% endstepper %}

***

#### 💡 샘플  불러오기 작성 팁

* 검증된 구조: 샘플 목차는 공시 기준을 충족하는 우수 사례를 바탕으로 합니다. 구조 잡기가 막막할 때 가이드라인으로 활용해 보세요.
* 산업 특화 지표 추가: 표준 목차는 보편적인 가이드를 제공하므로, 우리 업종만의 특수한 공시 항목이 누락되지 않았는지 점검하고 추가하세요.
* 자유로운 편집: 샘플은 말 그대로 '본보기'일 뿐입니다. 불러온 후에는 자유롭게 삭제하거나 순서를 바꿀 수 있으니 부담 없이 시작해 보세요.
