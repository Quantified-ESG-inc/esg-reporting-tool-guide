# \[도입] ERT 시작하기

## ESG 공시, 왜 해야 하나요?

* 규제 대응: 공시 의무화 및 글로벌 공급망 실사에 선제적으로 대응하기 위해 필요합니다.
* 이해관계자 소통: 기업의 비재무적 성과를 투명하게 공개하여 투자자와 고객의 신뢰를 확보합니다.



## ESG Reporting Tool (ERT)란 무엇인가요?

<mark style="color:$primary;">ERT(ESG Reporting Tool)</mark>는 기업의 복잡한 ESG 데이터를 체계적으로 관리하고, 글로벌 공시 기준에 부합하는 보고서를 신속하게 생성할 수 있도록 지원하는 <mark style="color:$primary;">ESG 웹공시 전문 솔루션</mark>입니다.

## 첫 ESG 보고서, PDF 대신  Web 보고서로 시작하세요.&#x20;

대부분의 기업이 ESG 보고서라고 하면 두꺼운 책자나 수백 페이지의 PDF 파일을 먼저 떠올립니다. 하지만 처음 시작하는 기업일수록 전통적인 PDF 방식은 실무자에게 큰 부담이 됩니다.

⚠️ 기존 PDF 보고서 발간의 한계

* 자원 과다 투입: 매년 발간할 때마다 새로운 표지와 내지 디자인을 개발해야 하므로 디자인 의사결정에 많은 시간과 비용이 소모됩니다.
* 수정의 어려움: 발간 후 오탈자가 발견되어도 이미 배포된 PDF 파일을 수정하거나 다시 배포하기가 매우 어렵고 시간이 오래 걸립니다.
* 연속성 결여: 작년에 만든 보고서를 활용하기 어렵고, 매년 처음부터 다시 디자인 외주를 맡겨야 하는 반복적인 비용이 발생합니다.



## ERT로 시작해야 하는 이유: "손쉬운 첫 보고서 발행"

ERT는 초보 담당자가 소수의 인력으로도 보고서 작성부터 유지·보수까지 관리할 수 있습니다.&#x20;

* **유지·보수의 편의성**: 웹상에서 오탈자를 직접 즉시 수정할 수 있으며, 수정 사항에 대해 담당자와 상시적인 피드백이 가능합니다.

&#x20;&#x20;

* **디자인 자산화**: 매번 새로운 디자인을 고민할 필요 없이, 검증된 메인 페이지 디자인을 유지하면서 내용만 업데이트할 수 있어 시간과 비용이 획기적으로 절감됩니다.



* **보고의 연속성 강화**: 작년 보고서 내용을 복제(Clone)하여 올해 지표로 업데이트하는 방식으로 작업할 수 있어, 차년도 보고의 효율성이 극대화됩니다.

&#x20;&#x20;



ERT는 초보 보고서 담당자가 겪는 3대 어려움을 해결합니다.

* **전문성 보완**: GRI, SASB 등 복잡한 표준을 몰라도 시스템이 제공하는 가이드를 따라가면 글로벌 수준의 공시가 가능합니다.



* **스토리텔링 해결**: 글쓰기가 막막한 담당자를 위해 AI 작성 도우미로 원고 작성을 지원합니다.



* **실시간 데이터 시각화**: 데이터를 입력하는 즉시 전문적인 그래프와 차트로 변환되어, 별도의 디자인 도구 없이도 고품질의 웹 보고서가 완성됩니다.





## ESG 보고서 발간 로드맵

일반적인 ESG 보고서 발간은 자료 취합부터 디자인까지 수개월의 긴 시간이 소요됩니다. 하지만 ERT(ESG Reporting Tool)는 시스템 자동화를 통해 이 과정을 혁신적으로 단축합니다.&#x20;
