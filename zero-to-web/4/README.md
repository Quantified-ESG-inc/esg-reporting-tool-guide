### 내부 자료를 활용하여 작성하기

보유하고 있는 사내 자료  또는   원고 초안을 AI 도우미에게 전달하여 보고서 원고를 자동 생성하는 방법입니다.

#### 1단계. 자료 업로드 및 AI 도우미  열기

* 자료 준비: 작성에 참고할 워드나 PPT 원고를 PDF로 변환하거나 이미지로 캡처하여 준비합니다.
* AI 도우미 실행: 화면 우측 상단의 AI 도우미 아이콘을 클릭한 뒤, 하단의 파일 선택 버튼을 눌러 준비한 자료를 첨부합니다.
* 문서 전체가 아닌 특정 도표나 그래프가 포함된 부분만 이미지로 캡처해 첨부해도 AI가 정확하게 데이터를 인식합니다.


{% endstep %}

{% step %}
#### 2단계. 분석 및 초안 생성 요청하기

* 요청사항 입력: 첨부한 파일에 대해 AI가 수행할 작업을 입력합니다.
  * _예: "첨부한 워드 파일의 내용을 기반으로 기후변화 대응 체계 원고를 작성해줘."_
  * _예: "이 PDF의 3\~4페이지 내용을 요약해서 보고서 본문에 넣어줘."_
* AI 생성: 입력 후 약 1분 내외의 시간이 지나면 AI가 자료를 분석하여 보고서 초안을 제안합니다.
* 신규 발간 기업의 경우, 아직 다듬어지지 않은 거친 상태의 초안이나 데이터 파일만 업로드해도 AI가 논리적인 보고서 문체로 다듬어 드립니다.
{% endstep %}

{% step %}
#### 3단계. 본문에 반영하기

* 본문에 추가: AI가 생성한 결과물을 확인하고, \[본문에 추가] 버튼을 눌러 즉시 보고서 섹션에 반영합니다.
{% endstep %}

{% hint style="warning" %}
#### 📌 꼭 확인해 주세요! (지원 형식 안내)

현재 AI 도우미의 원활한 분석을 위해 파일 업로드는 **PDF 및 이미지(PNG, JPG 등) 형태**만 지원하고 있습니다.

* 워드/PPT 자료 활용법: 보유하신 원고가 Word나 PPT 파일인 경우, 반드시 \[다른 이름으로 저장] → \[PDF]로 변환하여 업로드해 주세요.
* 이미지 캡처 활용법: 문서 전체가 아닌 특정 도표나 그래프 부분만 캡처(이미지 파일)하여 첨부하면 AI가 해당 데이터를 더욱 정확하게 인식하여 원고에 반영해 드립니다.
{% endhint %}

* 스마트 지원: 이 아이콘을 클릭하면 언제든지 스마트한 작성 지원을 받을 수 있는 준비가 완료됩니다.

* 본문에 추가: 생성된 내용이 마음에 든다면 \[본문에 추가] 버튼을 클릭하여 보고서 섹션에 즉시 반영합니다.
* 자유로운 수정: 본문에 추가된 AI 초안을 기반으로 우리 기업의 구체적인 수치나 세부 데이터를 보완하여 원고를 완성하세요.

* 결과물 검토: AI가 제안한 본문으로 수정한 후에는, 우리 기업의 실제 상황과 일치하는지 최종적으로 한 번 더 확인해 주는 것이 좋습니다.

## AI 생성 콘텐츠 이용 시 주의사항

AI 도우미는 여러분의 보고서 작성을 돕는 강력한 조력자이지만, 생성된 결과물이 곧바로 완벽한 최종안이 되는 것은 아닙니다. 보고서의 신뢰성을 지키기 위해 아래 내용을 반드시 확인해 주세요.



**1. 원고의 신뢰도와 검증 책임**

* 데이터 정합성 확인: 생성형 AI의 특성상 실제 사실과 다르거나 논리적 오류가 포함된 '환각 현상(Hallucination)'이 발생할 수 있습니다. 특히 수치 데이터나 지표 정보는 반드시 사내 데이터와 일치하는지 담당자가 직접 대조해야 합니다.
* 책임의 한계: AI가 생성한 원고를 검토 없이 그대로 사용하여 발생하는 보고서의 오류나 신뢰도 저하에 대해 QESG는 책임을 지지 않습니다. 최종 보고서에 담긴 모든 내용에 대한 책임은 해당 기업과 작성 담당자에게 있습니다.



**2. 담당자의 최종 수정 및 검토 필수**

* 전문가 가이드로서의 활용: AI가 제안한 내용은 원고 작성을 위한 '초안' 또는 '가이드라인'으로 간주해 주세요.
* 담당자 검토 필수: AI로 작성한 내용을 바탕으로 우리 기업만의 구체적인 사례와 최신 성과를 덧붙여 원고를 완성해야 합니다. 재작성 기능을 활용해 다듬더라도, 최종 발행 전에는 **반드시 담당 부서의 승인 및 검토 절차**를 거치시길 권장합니다.
{% endhint %}
