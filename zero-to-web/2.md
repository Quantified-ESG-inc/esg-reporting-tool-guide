1단계. 새 목차 생성 시작하기

1. 목차 작성 방식 선택 화면에서 두 번째 항목인 \[새 목차 생성] 카드를 클릭합니다.
2. 하단의 \[체험 시작] 버튼을 눌러 편집 화면으로 이동합니다.

2단계. 대분류 및 소분류 구성하기

편집 화면에 진입하면 본격적으로 보고서의 뼈대를 잡을 수 있습니다.

1. 목차 추가하기: 화면 중앙의 \[+ 목차 추가하기] 버튼을 클릭하여 첫 번째 목차 항목을 생성합니다.
2. 항목 입력: \* 대분류: 보고서의 큰 장(Chapter) 제목을 입력합니다. (예: 환경 경영, 사회적 책임 등)
3. 소분류: 해당 장에 포함될 세부 항목을 입력합니다. (예: 온실가스 배출량 관리, 안전보건 정책 등)
4. 소분류 추가: 소분류 입력창 아래의 \[+] 버튼을 누르면 해당 대분류 안에 더 많은 소분류 항목을 계속해서 추가할 수 있습니다.

3단계. 목차 확장 및 저장하기

1. 대분류 추가: 하나의 대분류 작성이 끝났다면, 하단의 \[+ 목차 추가하기] 버튼을 다시 눌러 다음 대분류를 생성합니다.
2. 목차 저장: 모든 목차 구성이 완료되었다면 화면 좌측 하단의 \[목차 저장하기] 버튼을 클릭하여 설정을 완료합니다.
